# HeartLink Synthetic Validation — Trial 2025-10-LLC_Baseline

## 1. Purpose
This trial establishes the baseline equivalence between HeartLink algorithm versions v3.6 and v3.7 using 15 synthetic clinical scenarios.  
It represents the first validation trial under HeartLink LLC following legal formation.
